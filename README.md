# CSC 299 Final Project – AI-Coding Assistant System

## Overview
This project implements:
- **PKMS**: a personal knowledge management system  
- **Task Manager**: create, list, search tasks  
- **Chat Interface**: terminal-based interaction  
- **AI Agents**: analyze or summarize tasks  

## Directory Structure
```
tasks1/ – first prototype  
tasks2/ – second iteration  
tasks3/ – pytest + packaging version  
tasks4/ – (planned) OpenAI Chat Completions API demo  
docs/   – reflections & saved conversations
```

## How to Run
```bash
uv run tasks3
```

## How to Test
```bash
cd tasks3
uv run pytest
```

## Requirements
- Python 3.11 or newer  
- [uv](https://docs.astral.sh/uv/)  
- pytest  

Runs on macOS, Windows, and Linux.

## Deliverables
- Functional PKMS + Task Manager + Chat Interface  
- AI Agent integration (task summarization or planning)  
- `SUMMARY.md` (development reflection ≥ 500 words)  
- `video.txt` (YouTube link to 6–8 min demo)
